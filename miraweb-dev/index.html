MIRA

Directions: Just talk normal.

To my knowledge Mira is the most lightweight full-featured agent available today. I firmly believe agents don't need overt complexity when built on a strong foundation with reliable tool and memory integration. MIRA's flexible architecture enables contextual tool discovery, usage, and deactivation, with memory that adapts to user preferences over time. The just-in-time discovery approach makes MIRA fast.

Some (read: all) agent systems throw compute at the problem, Manus has a 500 line system message. Mira's system message at startup is 6 declarative sentences.
